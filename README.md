# DMS qmldir files

For some reason DMS (dankmaterialshell) doesn't include qmldir files on their project, which makes plugin dev using qmlls near impossible. So here are some qmldir files I made

The provided script can be run from nushell to autogenerate all qmldir files.
It must be run with the cwd set to the root of dms.

If this repo ever gets out of date, it is still trivial to get a working copy thanks to the script
